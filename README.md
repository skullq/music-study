# 통합 음악 시각화 웹 서비스 (Music Visualization Web Service)

사용자가 선택한 스케일(음계)과 코드(화음)를 **오선지, 피아노 건반, 기타 지판(Fretboard)** 에 한꺼번에 시각화하여 보여주는 인터랙티브 웹 애플리케이션입니다. 

복잡한 음악 이론(다이아토닉, 분수코드, 텐션 등)을 눈으로 보고, 직접 마우스로 클릭하여 Web Audio API로 소리를 들으며 직관적으로 학습할 수 있도록 설계되었습니다.

<br/>

## 주요 기능 (Key Features)

* **다중 악기 동시 시각화**: 한 번의 조작으로 오선지 악보, 2옥타브 피아노 건반, 6번 줄 15프렛 기타 지판에 스케일과 코드가 동시 매핑됩니다.
* **음악 이론 구조화**: 
  * 스케일 렌더링 시 **온음 / 반음 간격**을 오선지 악보 아래에 직관적인 선으로 그려줍니다.
  * 스케일 모드에서 3화음/7화음 다이아토닉 코드를 쌓아 볼 수 있으며, 화성학 로마숫자(I, ii, V7 등)와 실제 코드 네임이 함께 표기됩니다.
  * 9, 11, 13화음과 서스펜디드, Add, 그리고 베이스 음이 바뀌는 **분수코드(Slash Chord)**까지 완벽하게 지원합니다.
  * 기타 지판에서 **근음(Root)** 을 붉은색으로 강조하여 스케일 폼을 익히기 쉽게 돕습니다.
* **실시간 인터랙션 & 사운드**: 
  * 오선지, 피아노, 기타 지판의 음표를 클릭하면 **Web Audio API** 기반의 내장 신디사이저로 소리가 재생됩니다.
  * 코드 블록을 클릭하면 모든 음이 동시에, 단일음을 클릭하면 아르페지오 형식으로 재생되며 연주 중인 음표가 빨간색/파란색으로 **실시간 하이라이팅** 됩니다.

## 🛠 기술 스택 (Tech Stack)

**Backend**
* Python 3.x
* **FastAPI**: 빠르고 가벼운 RESTful API 서버 구축
* **music21**: 복잡한 12음계, 스케일 및 화음 구성음 매핑 연산 전담

**Frontend**
* **Vanilla JS**: 상태 관리 및 DOM 제어
* **VexFlow**: 오선지 악보 및 임시표, 간격 기호 SVG 렌더링
* **Web Audio API**: 브라우저 내장 오디오 신디사이저 구현

## 🚀 설치 및 실행 방법 (How to Run)

본 프로젝트는 파이썬 패키지 관리자인 `uv` (또는 `pip`)를 기반으로 실행됩니다.

1. **가상환경 생성 및 의존성 패키지 설치**
   ```bash
   uv venv
   uv pip install fastapi uvicorn music21
   ```

2. **FastAPI 서버 실행**
   ```bash
   uv run uvicorn app.main:app --reload
   ```

3. **웹 브라우저 접속**
   서버가 실행되면 웹 브라우저를 열고 아래 주소로 접속합니다.
   ```text
   http://localhost:8000
   ```

## 💡 사용법 (How to Use)

1. **기본 설정:** 화면 상단의 첫 번째 드롭다운 메뉴에서 **근음(Root Note)** 을 선택합니다. (예: C, F#, Bb 등)
2. **카테고리 선택:** 두 번째 메뉴에서 학습할 카테고리를 선택합니다. (스케일, 3화음, 7화음 ~ 분수코드 등)
3. **상세 타입 선택:** 세 번째 메뉴에서 세부 코드를 선택합니다. (예: Major, Minor 7th, Sus4, C/E 등)
4. **화음 쌓기 (스케일 모드 전용):** 스케일 카테고리 선택 시 활성화되는 메뉴로, 스케일 위에 3화음 또는 7화음(다이아토닉)을 한 번에 쌓아볼 수 있습니다.
5. **검색 옵션 (코드 모드 전용):** 기타 지판에서 '대표 운지법 하나만 볼지', '가능한 모든 운지법을 볼지' 선택합니다.
6. **렌더링:** **[렌더링]** 버튼을 클릭하면 세 가지 악기에 음악 이론이 즉시 시각화됩니다.
7. **소리 듣기 (인터랙션):**
   * **오선지:** 앞쪽 덩어리(블록) 코드를 클릭하면 화음이 동시에 쳐지고, 뒤에 펼쳐진 개별 음표를 클릭하면 단일음/아르페지오로 연주됩니다.
   * **피아노:** 원하는 건반을 마우스로 클릭하여 자유롭게 연주합니다.
   * **기타:** 동그란 마커(운지 위치)를 클릭하여 각 줄의 소리를 들어볼 수 있습니다.
