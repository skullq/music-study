# 🎵 통합 음악 시각화 웹 서비스 (Music Visualization Web Service)

사용자가 선택한 스케일(음계)과 코드(화음)를 **오선지, 피아노 건반, 기타 지판(Fretboard)** 에 한꺼번에 시각화하여 보여주는 인터랙티브 웹 애플리케이션입니다. 
음악 이론을 눈으로 보고, 직접 클릭하여 소리를 들으며 직관적으로 학습할 수 있도록 돕습니다.

<br/>

## ✨ 주요 기능 (Key Features)

* **🎼 다중 악기 동시 시각화**: 한 번의 조작으로 오선지 악보, 2옥타브 피아노 건반, 6번 줄 15프렛 기타 지판에 스케일과 코드가 동시 매핑됩니다.
* **🎸 음악 이론 구조화**: 
  * 스케일 렌더링 시 **온음(W) / 반음(H) 간격**을 오선지와 피아노 건반 위에 직관적인 선으로 그려줍니다.
  * 화음(Triad) 렌더링 시 **블록 코드(Block Chord)** 와 **로마 숫자(I, III, V)** 를 악보에 함께 표기합니다.
  * 기타 지판에서 **근음(Root)** 을 붉은색으로 강조하여 스케일 폼을 익히기 쉽게 돕습니다.
* **🎹 실시간 인터랙션 & 사운드**: 
  * 피아노 건반이나 기타 지판의 음표를 마우스로 클릭하면 **Web Audio API**를 통해 실제 피아노 소리가 재생됩니다.
  * 소리가 나는 동안 오선지 악보의 해당 음표가 빨간색으로 **실시간 하이라이팅** 되어 시각과 청각을 완벽하게 연결합니다.

<br/>

## 🛠 기술 스택 (Tech Stack)

**Backend**
* Python 3.x
* **FastAPI**: 빠르고 가벼운 RESTful API 서버 구축
* **music21**: 복잡한 12음계, 스케일 및 화음 구성음 매핑 연산 전담

**Frontend**
* HTML5 / CSS3 / Vanilla JS
* **VexFlow**: 오선지 악보 및 임시표, 간격 기호 SVG 렌더링
* **Web Audio API**: 브라우저 내장 오디오 신디사이저 구현

<br/>

## 🚀 설치 및 실행 방법 (How to Run)

본 프로젝트는 파이썬 패키지 관리자인 `uv` (또는 `pip`)를 기반으로 실행됩니다.

1. **가상환경 생성 및 의존성 패키지 설치**
   ```bash
   uv venv
   uv pip install -r requirements.txt
   ```

2. **FastAPI 서버 실행**
   ```bash
   uv run uvicorn app.main:app --reload
   ```

3. **웹 브라우저 접속**
   서버가 실행되면 웹 브라우저를 열고 아래 주소로 접속합니다.
   ```text
   http://localhost:8000
   ```

<br/>

## 💡 사용법 (How to Use)

1. 화면 상단의 **드롭다운 메뉴**에서 원하는 **근음(Root Note)** 과 **스케일/코드 종류**를 선택합니다. (예: C, Major Triad)
2. **[렌더링]** 버튼을 클릭합니다.
3. 하단에 생성된 오선지, 피아노, 기타 지판 뷰를 확인하며 음악 이론을 학습합니다.
4. 피아노의 건반이나 기타 지판의 파란색/빨간색 동그라미를 **클릭하여 직접 연주**해 봅니다.
